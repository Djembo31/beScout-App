# Slice 148b — Gençlerbirliği Logo Fix (XS data-fix)
Date: 2026-04-22

## Trigger

Anil observation: DB `logo_url = https://media.api-sports.io/football/teams/997.png` zeigt falsches Wappen für Gençlerbirliği S.K.

## Root-Cause Hypothesis

api-football team 997 shows a crest that does NOT match Gençlerbirliği's official club colors (black field + red crescent + Hittite sun + 1923 text on white outer ring). Likely api-sports team 997 is a different Turkish club or api-sports has wrong image for this ID.

## Source (Anil)

Anil provided: https://genclerbirligi.org.tr/haberler/kis-donemi-2-futbol-okullari-kayitlari-devam-ediyor-25-aralik-2025 (official club website). Direct fetch blocked (403), fell back to Wikipedia canonical crest.

## Fix

```sql
UPDATE clubs
SET logo_url = 'https://upload.wikimedia.org/wikipedia/en/thumb/1/13/Gen%C3%A7lerbirli%C4%9Fi_S.K._crest.svg/250px-Gen%C3%A7lerbirli%C4%9Fi_S.K._crest.svg.png',
    updated_at = NOW()
WHERE id = 'cb174221-b3da-46a8-98e4-6d448a0973ae';
```

1 row affected.

## CSP / Next-Image Whitelist Verified

- `vercel.json` img-src includes `https://upload.wikimedia.org`  ✓
- `next.config.mjs:29` hostname `upload.wikimedia.org`  ✓

Matches Gençlerbirliği's official identity (rot-schwarz, Ankara, est. 1923).
