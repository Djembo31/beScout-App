## F4 Market-Tabs A11y (aria-selected + role=tab/tablist)

Before:
  src/features/market/components/MarketContent.tsx:155-170
  <div className="flex...">
    {tabs.map(tb => <button onClick={...}>{tb.label}</button>)}
  </div>
  // no role, no aria-selected, no aria-controls

After:
  <div role="tablist" aria-label="Market-Bereiche" className="flex...">
    {tabs.map(tb => (
      <button
        role="tab"
        id={`tab-${tb.id}`}
        aria-selected={isActive}
        aria-controls={`tabpanel-${tb.id}`}
        tabIndex={isActive ? 0 : -1}
        className="... focus-visible:ring-2 focus-visible:ring-gold/50 ..."
      >{tb.label}</button>
    ))}
  </div>

IDs match existing TabPanel component (tabpanel-${id} + tab-${id}).
Screen-Reader hears "tab, selected, Mein Kader" instead of just "button Mein Kader".
Keyboard focus-indicator added (focus-visible ring).
