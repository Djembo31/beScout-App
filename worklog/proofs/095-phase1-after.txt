## Changes Summary

### 1. DB Migration (applied via MCP)
Migration `slice_095_trade_projection_rpcs`:
- `get_player_trade_history(p_player_id, p_limit)` — SECURITY DEFINER, handle+is_own projection

### 2. Type `PublicTrade` (src/types/index.ts)
Neuer type parallel zu `DbTrade`:
- Keine `buyer_id` / `seller_id` (raw exposure eliminiert)
- Neu: `buyer_handle`, `seller_handle` (LEFT JOIN profiles)
- Neu: `is_buyer_own`, `is_seller_own` (auth.uid() compare)
- Neu: `is_ipo_buy` (computed: ipo_id NOT NULL OR seller_id IS NULL)

### 3. Service Migration (src/lib/services/trading.ts)
- `getPlayerTrades` → RPC `get_player_trade_history`, returns PublicTrade[]
- `getAllPriceHistories` → RPC `get_global_price_sparkline`

### 4. UI Migration
- `TradingTab.tsx` — buyer_handle/seller_handle/is_buyer_own/is_seller_own statt buyer_id/seller_id lookup
- `YourPosition.tsx` — filter auf is_buyer_own (userId nicht mehr nötig für filter-logic)
- `PriceChart.tsx`, `TradingQuickStats.tsx`, `CommunityTab.tsx` — type `DbTrade[]` → `PublicTrade[]`
- `usePlayerDetailData.ts` — profileMap-auto-populate-Effect entfernt (trades carrying handles now)

### 5. Tests
