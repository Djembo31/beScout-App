Identified during event ownership brainstorming (2026-03-25). These are validated revenue opportunities to build after the event ownership system is complete.

**Why:** Business analyst + competitor research identified gaps in BeScout's monetization that competitors already exploit.

**How to apply:** Prioritize these in future sprints after event ownership system ships.

## 1. Sponsor Flat Fee (HIGH PRIORITY)
- Sponsors pay EUR directly (B2B, off-platform) to brand an event
- Logo, name, custom branding on event cards
- Pure margin for BeScout + hosting Club
- Range: 500-5,000 EUR per sponsored event depending on reach
- Entry fee split stays normal — sponsorship fee is on top

## 2. Event Boost / Featured Placement (MEDIUM)
- Club pays extra $SCOUT/EUR to get event featured on homepage
- Push notifications to all users for boosted events
- Tiered: Basic boost (homepage) / Premium boost (homepage + push + email)

## 3. Chip/Power-Up Economy (MEDIUM)
- Triple Captain, Bench Boost, Wildcard etc. (FPL-proven model)
- Purchasable for Tickets or $SCOUT
- Direct revenue stream tied to events
- Codebase already has `chip_use` and `chip_refund` as ticket sources

## 4. B2B Event-as-a-Service Overage (MEDIUM)
- Included in club sales packages:
  - Baslangic: 4 Events/Monat
  - Profesyonel: 12 Events/Monat
  - Sampiyon: Unlimited
- Overage charges for events beyond package limit

## 5. Event Analytics & Data Licensing (LONG TERM)
- Fan sentiment data: which players fans pick, formation trends
- Valuable to: Clubs (player perception), Agents (market data), Media (content)
- Referenced in VISION.md but no fee structure yet

## 6. Event Replay/Post-Game Insights (LONG TERM)
- "Your lineup vs winner's lineup", "Players you missed"
- Behind soft paywall (1-2 tickets for full breakdown)
- Drives engagement + small revenue

## 7. Guaranteed Prize Pool Subsidies (GROWTH)
- Platform guarantees minimum prize pool for club events
- Removes club risk, builds confidence
- DraftKings did this at launch — ran events at a loss to build habits
- Cost center, not revenue — but critical for cold-start
