# Wiki-Lint Checklist

> AutoDream runs this checklist during consolidation passes.
> Findings get written into the "Last Run" section below.

## Checks

### Contradictions
- Pages that claim opposite things about the same topic?
- session-handoff.md vs current-sprint.md inconsistencies?
- Feature specs that don't match actual code state?

### Stale Claims
- Pages referencing features/files that no longer exist?
- Status claims ("in progress") that are actually done or abandoned?
- Error patterns that have been superseded by newer patterns?

### Orphan Pages
- Files in memory/ with no references from cortex-index or wiki-index?
- Feature specs for features that were never built or were removed?

### Missing Concepts
- Topics in retros/handoffs that lack their own wiki page?
- New domains (e.g., "offer system", "mystery box economy") without semantic pages?

### Cross-Reference Gaps
- cortex-index routing entries pointing to deleted/moved files?
- wiki-index entries with wrong line counts or descriptions?
- common-errors.md patterns that should be cross-referenced in database.md or trading.md?
