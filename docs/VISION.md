## Was ist BeScout?

**BeScout ist für Fußball-Wissen, was YouTube für Gesang und TikTok für Kreativität ist.**

Auf Social Media wirst du berühmt, weil du gut singst, lustig bist, gut schreibst. Auf BeScout wirst du berühmt, weil du **Fußball verstehst** — und du wirst dafür **bezahlt**.

BeScout ist eine **B2B2C Fan-Engagement- und Monetarisierungsplattform** für Fußballvereine.

- **B2B:** Vereine nutzen BeScout als Tool, um Fans aufzubauen, zu binden und zu monetarisieren.
- **B2C:** Fans kaufen Spielerkarten, spielen Fantasy, schreiben Analysen, stimmen ab — und verdienen BSD.
- **Plattform:** BeScout stellt die Infrastruktur und nimmt eine Plattform-Gebühr.

## Das Problem

```
Galatasaray: 30M+ Follower auf Social Media

├── Twitter/X:  Millionen Fan-Diskussionen → Twitter verdient
├── YouTube:    Millionen Views auf Fan-Content → YouTube verdient
├── Instagram:  Millionen Kommentare → Meta verdient
│
├── Galatasaray verdient: 0 €
└── Fans werden gehört: Nie
```

Fans investieren Zeit, Leidenschaft, Wissen — aber der Verein hat nichts davon, und die Fans haben keine Stimme.

## Die Lösung

BeScout kanalisiert Fan-Engagement, das heute wertlos auf Social Media verpufft, in eine Plattform wo **jede Interaktion dem Verein Einnahmen bringt und dem Fan eine echte Stimme gibt**.

```
Fan-Aktivität heute (gratis, wertlos)     →   BeScout (monetarisiert, zählt)
──────────────────────────────────────────────────────────────────────────────
"Müsli sollte in die Startelf!"           →   Fantasy Lineup (Entry Fee)
"Wir sollten einen LV holen!"             →   Club Vote (BSD-Kosten)
"Icardi ist überschätzt"                  →   DPC Sell Order (Trading)
"Barış Alper wird Star"                   →   DPC Kaufen / IPO (Investment)
"Schaut meine Analyse!"                   →   Bericht verkaufen (Paywall)
Twitter-Poll: "Bester Spieler?"           →   Bezahlte Poll (BSD)
```

## Fan-Ökonomie: Die 7 Verdienst-Wege

### 1. Trading (Live)
Günstig kaufen → teuer verkaufen. Dein Wissen über Spieler = dein Vorteil.

### 2. Fantasy Turniere (Live)
Beste Aufstellung → BSD gewinnen. Beweist: du kannst Spieler bewerten.

### 3. Success Fee + PBT (Geplant)
Halte DPCs langfristig → werde belohnt bei Vertragsablauf.
- **PBT (Player Bound Treasury):** Topf pro Spieler, füllt sich durch Trading-Gebühren. Bei Liquidierung proportional an DPC-Holder ausgeschüttet.
- **Success Fee:** Vom Verein freiwillig gezahlt, mit Cap. Verein entscheidet ob und wann.
- Beweist: du erkennst Talent früh.

### 4. Berichte verkaufen (Geplant)
Schreibe Analyse zu Spieler/Verein → verkaufe an andere User für BSD.
Je besser dein Track Record → mehr Käufer → mehr Einnahmen.

### 5. Club-Aufträge / Bounties (Geplant)
Verein stellt Auftrag ein → Fans liefern Analyse → Verein bezahlt in BSD.
Fan bekommt: BSD + Reputation + Sichtbarkeit beim Club.

### 6. Polls / Paywall / Premium Content (Geplant)
- Bezahlte Polls erstellen (Fans zahlen BSD zur Teilnahme)
- Premium-Beiträge hinter Paywall (BSD zum Lesen)
- Predictions: bewiesene Vorhersagen = Reputation

### 7. Follower-Monetarisierung (Geplant)
Baue Reichweite auf → monetarisiere deine Audience.
- Kostenlose Beiträge (Reichweite aufbauen)
- Premium-Beiträge (nur für Follower / gegen BSD)
- Club-Abos (Fans abonnieren dich für X BSD/Monat)

## Fan-Identität: Dein Fußball-CV

Jede Aktion auf BeScout baut ein **verifiziertes Profil** auf.

Auf Twitter kann jeder behaupten "Ich hab's gewusst". Auf BeScout sieht man:
- Wann du gekauft/verkauft hast (Trading History)
- Wie deine Fantasy-Aufstellungen abgeschnitten haben (Ergebnisse)
- Ob deine Berichte gut bewertet wurden (Ratings)
- Ob Clubs dich beauftragt haben (Aufträge)

**Kein Bluffen möglich. Dein Track Record ist dein Ruf.**

### Profil-Bausteine:
- **Scout Score:** Trading (35%) + Manager (40%) + Scout (25%)
- **Level & Tier:** Rookie → Amateur → Profi → Elite → Legende → Ikone
- **Achievements:** 18+ verifizierte Meilensteine
- **Track Record:** Verifizierte Trading/Fantasy/Analyse-History
- **Follower & Reichweite:** Deine Community
- **Club-Aufträge:** Professionelle Referenzen
- **Missions:** Tägliche/wöchentliche Aktivitätsziele

### Der ultimative Anreiz:
Herausragende Fans werden von Clubs für **echte Positionen** entdeckt — Scouting, Analyse, Community Management. BeScout wird zum Talent-Pool für die Fußball-Industrie.

## Vereins-Ökonomie: Revenue-Ströme

| Quelle | Beschreibung |
|--------|-------------|
| **DPC-Verkäufe (IPOs)** | Verein verkauft Spielerkarten → direkte Einnahmen |
| **Trading-Gebühren** | Jeder Sekundärmarkt-Trade → Anteil für Verein |
| **Vote-Einnahmen** | Fans zahlen BSD um abzustimmen |
| **Event Entry Fees** | Fantasy Events → Verein erstellt, Fans zahlen Entry |
| **Mitglied-Abos** | Premium-Zugang (Early Access IPOs, exklusive Votes) |
| **Fan-Daten & Insights** | Verein weiß wer seine aktivsten Fans sind |
| **Bounty-System** | Verein beauftragt Fan-Analysen → zahlt für Qualität |

### Fee-Split bei Trades:
```
Trade: 10 BSD
  ├── X% Gebühr (konfigurierbar)
  │   ├── Y% → PBT des Spielers
  │   ├── Z% → BeScout Plattform
  │   └── W% → Club Revenue
  └── Rest → Verkäufer
```

### Success Fee Modell:
```
Vertragsablauf eines Spielers:
  ├── PBT wird ausgeschüttet (automatisch, immer)
  │   └── Proportional zu DPC-Besitz
  ├── Success Fee (OPTIONAL, vom Verein gesteuert)
  │   ├── Verein setzt Cap (z.B. max 50 BSD/DPC)
  │   └── Verein entscheidet ob und wann
  └── DPCs werden liquidiert (ungültig)
```

## Der Pitch

### An Vereine:
> "Ihr habt 30 Millionen Follower. Die diskutieren jeden Tag eure Aufstellung, eure Transfers, eure Strategie. Twitter verdient daran — ihr nicht. Und eure Fans? Die fühlen sich ignoriert.
>
> BeScout ändert das. Eure Fans kaufen Spielerkarten, stellen Lineups auf, stimmen über echte Club-Entscheidungen ab — und ihr verdient an jeder Interaktion. Gleichzeitig wisst ihr zum ersten Mal, was eure Fans wirklich denken.
>
> Nicht noch ein Social Media Kanal. Sondern der erste Kanal, der euch gehört."

### An Fans:
> "Du diskutierst sowieso jeden Tag über Fußball. Auf Twitter interessiert das niemanden. Auf BeScout wirst du dafür bezahlt.
>
> Kaufe Spielerkarten, stelle dein Dream-Team auf, schreibe Analysen. Dein Wissen hat hier echten Wert — in BSD, in Reputation, und vielleicht sogar in einem echten Job beim Club."
