# decisions/ — warum (ADRs)

Begründete Entscheidungen mit Alternativen-Abwägung: **warum** dieser Weg, was wurde erwogen, Re-Visit-Trigger.

**SSOT bleibt `memory/decisions.md`** (chronologisches Log D1–D87). Dieser Ordner ergänzt nur dann, wenn ein Thema eine eigene, gepflegte Seite verdient (z. B. ein vielschichtiger Money-/Architektur-Entscheid, der über mehrere D's gewachsen ist). Dann: Seite hier + Zeiger aus den betroffenen D-Einträgen, keine stille Kopie.

**Gehört hierher:** thematisch verdichtete ADRs (Treasury-Architektur-Warum, Pricing-Asset-Modell-Warum).

**Gehört NICHT hierher:** einzelne chronologische D-Einträge (bleiben in `decisions.md`) · Mechanik (`domain/`).
